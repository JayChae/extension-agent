# AI 신입사원

법원 전자소송 사이트(ecfs.scourt.go.kr) 업무를 자동화하는 **크롬 익스텐션 + 백엔드**입니다.
아래 순서대로 따라 하면 내 컴퓨터에서 실행해볼 수 있습니다.

> 설계 문서는 [_docs/architecture.md](_docs/architecture.md), 개발 순서는 [_docs/TODO.md](_docs/TODO.md)를 보세요.

## 준비물 (한 번만)
- 이 폴더(코드)를 내려받아 둡니다.
- `uv`가 설치되어 있어야 합니다. (없으면 개발자에게 문의)

## 1단계 — 비밀 키 넣기
1. `backend` 폴더 안의 `.env.example` 파일을 복사해, 같은 폴더에 `.env` 라는 이름으로 붙여넣습니다.
2. `.env` 파일을 열어 `ANTHROPIC_API_KEY=` 뒤에 발급받은 키를 붙여넣고 저장합니다.

> ⚠️ 이 키는 절대 다른 사람과 공유하거나 깃(git)에 올리지 마세요.
> (`.env` 파일은 자동으로 git에서 제외되도록 설정돼 있습니다.)

## 2단계 — 백엔드 켜기
1. 터미널을 엽니다.
2. 아래 명령을 한 줄씩 복사해 붙여넣고 엔터합니다.

   ```
   cd backend
   uv run uvicorn main:app --reload
   ```

3. `Uvicorn running on http://127.0.0.1:8000` 같은 글자가 보이면 성공입니다.
   - 웹브라우저 주소창에 `http://127.0.0.1:8000` 을 치면 `{"status":"ok"}` 가 보입니다.
   - 끄려면 터미널에서 `Ctrl + C` 를 누릅니다.

## 3단계 — 크롬에 익스텐션 올리기
1. 크롬 주소창에 `chrome://extensions` 를 입력하고 엔터합니다.
2. 오른쪽 위 **개발자 모드(Developer mode)** 스위치를 켭니다.
3. 왼쪽 위 **압축해제된 확장 프로그램을 로드합니다(Load unpacked)** 버튼을 누릅니다.
4. 이 프로젝트의 `extension` 폴더를 선택합니다.
5. 목록에 **AI 신입사원** 이 나타나면 성공입니다.
6. 크롬 툴바의 확장 아이콘을 누르면 오른쪽에 사이드패널이 열립니다.

## 4단계 — 메시지가 한 바퀴 도는지 확인 (Phase 2)

> 아직 "두뇌(AI)"는 없습니다. **글자가 사이드패널 → 백엔드 → 화면(content script) → 다시 사이드패널로 한 바퀴 돌아오는 길**만 확인하는 단계입니다.

1. 위 2단계대로 백엔드를 켜 둡니다.
2. 크롬에서 `https://ecfs.scourt.go.kr` 같은 **법원 전자소송 사이트 탭**을 하나 엽니다. (이 사이트에서만 동작하도록 막아 두었습니다.)
3. 그 탭이 **화면에 보이는 상태**에서, 툴바의 확장 아이콘을 눌러 오른쪽 사이드패널을 엽니다.
   - 패널 위에 **"연결됨"** 이라고 뜨면 백엔드와 잘 연결된 것입니다.
4. 아래 입력칸에 아무 글자(예: `안녕`)나 적고 **보내기**를 누릅니다.
5. 패널에 아래 3줄이 차례로 보이면 성공입니다.
   - `안녕` (내가 보낸 말)
   - `[backend] 안녕` (백엔드를 거쳐 돌아옴)
   - `[content@https://...scourt.go.kr/...] 안녕` (법원 페이지가 받아서 응답)
6. 빨간 **STOP** 버튼을 누르면 입력이 멈추고 "정지됨"이 표시됩니다. (안전장치 자리)

> 법원 탭이 안 보이는 상태에서 보내면 "활성 scourt 탭이 없습니다" 안내가 뜹니다 — 법원 탭을 켜고 다시 시도하세요.

## 폴더 설명
- `backend/` — 두뇌·금고 (Claude 판단 + 비밀 키). API 키는 **여기에만** 있습니다.
- `extension/` — 크롬 익스텐션 (사이드패널 + content script).
- `memory/` — 배운 내용(SOP·노트) 저장소. (지금은 비어 있음)
- `_docs/` — 설계 문서.
